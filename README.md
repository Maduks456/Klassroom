# Klassroom
## About
<p>
    Its a classroom webpage where its easy for teachers to make a class and give homework 
</p>
<p>
    Built with Laravel and breeze.
</p>

## Features
<ul>
    <li>
        Give accounts a profile picture (That broke when putting the project to vercel)
    </li>
    <li>
        Make classes (Teacher)
    </li>
    <li>
        Give/send homework (Teacher/Pupil)
    </li>
    <li>
        Rate homework (Teacher)
    </li>
    <li>
       Edit accounts (Admin)
    </li>
    <li>
       See action logs (Admin)
    </li>
</ul>
